    Getting Started with SpadeBloom
    Production Control System — v1.0

    1. Overview

    SpadeBloom's Production Control System (PCS) is a full-stack Manufacturing Execution System designed for modern factory operations. It provides real-time dashboards, work order management, operator tracking, and built-in analytics — all in one platform.

    2. System Requirements

    • Modern web browser (Chrome, Firefox, Safari, Edge — latest 2 versions)
    • Stable internet connection (minimum 5 Mbps recommended)
    • Mobile app available for iOS 15+ and Android 12+
    • Minimum screen resolution: 1280x720 (desktop), 375px width (mobile)

    3. Quick Start Guide

    Step 1: Sign up at spadebloom.com and create your organization workspace.

    Step 2: Add your products to the Product Catalogue with SKU numbers and descriptions.

    Step 3: Define manufacturing processes — assign stages, set target durations, and link to products.

    Step 4: Create work orders and assign operators to each stage.

    Step 5: Monitor progress in real-time via the Dashboard and Kanban views.

    4. Key Modules

    Dashboard — Real-time KPI tiles, work order status donut chart, and live operator tracking table.

    Product Catalogue — Manage all manufactured products with icons, SKUs, descriptions, and pagination.

    Process Designer — Create and version manufacturing workflows with ordered stages and color-coded progress.

    Work Orders — Full lifecycle management with stage-by-stage breakdowns, operator assignments, and alerts.

    Kanban Pipeline — Visual drag-and-drop board with Pending, In Progress, Completed, and Skipped columns.

    Time Tracking — Live elapsed time per operator with clock in/out and session history.

    Reports & Analytics — Operator performance charts, stage analytics, and CSV export with custom date ranges.

    3D Quality Analysis — Upload 3D models or CAD files for real-time quality inspection with point cloud scanning.

    5. User Roles & Permissions

    • Admin — Full access to all modules, user management, and system settings
    • Manager — Access to dashboards, reports, work orders, and team oversight
    • Supervisor — Manage operators, correct time entries, and monitor floor activity
    • Operator — Clock in/out, view assigned stages, and update work order progress

    6. API & Integrations

    SpadeBloom provides a RESTful API for custom integrations. WebSocket connections deliver real-time updates to all connected clients. The platform supports integration with ERP systems, IoT sensors, and third-party analytics tools.

    7. Support

    For technical assistance, reach out to support@spadebloom.com or visit our Contact page to raise a support ticket. Our team typically responds within 24 hours.
